LONGITUDINAL PROTON STRUCTURE FUNCTION DATA
==================================================

This directory contains FL (longitudinal structure function) data
separated by experiment from various deep inelastic scattering measurements.

COLUMN DESCRIPTIONS:
--------------------
Q^2      : Four-momentum transfer squared (GeV²)
xB       : Bjorken x (dimensionless)
W^2      : Invariant mass squared (GeV²)
FL       : Longitudinal structure function
errFL    : Statistical error on FL
experiment: Source experiment/collaboration
